Fun with Variables

We can add two numbers with +

    teddysAge = 10;
    myAge = 22;
    ourAge = myAge + teddysAge

We can join two strings with +, too!

    firstName = "Teddy"
    food = "Brocolli"
    sentence = firstName + " likes " + food

Careful! We can't join numbers and strings with +!

    firstName = "Teddy"
    age = 10
    sentence = firstName + " is " + age # ERROR!

if, else

Sometimes, we want to do something only if something else is true!

Example: we want to turn on the light until it is too bright, or we want to keep looking though a stack of numbers until we find the biggest one (which means, until we get to the end of the stack)

How do we check this in python?

    def makeBrighter(light): # light is a NUMBER
      if light < 100:
	light = light + 1
      return light

We can do some logical math to figure out if things are true or not

    name = "Akiva" # name is now Akiva
    if name == "Teddy": # false!
       print "What's up, Teddy"
    else:
       print "You're super cool."

for, while

Sometimes, we want to do something many times

example making random names: add letters until the length of the name is long enough

We can use while to check that a logical condition is still being kept!

    from random import randint
    name = "" # no name yet
    ordA = ord('A')
    ordZ = ord('Z')
    
    while len(name) < 10:
	name = name + randint(ordA, ordZ)
    return name
